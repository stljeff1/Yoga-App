This website, first and foremost is a demonstration and experiement in using AngularJS to build a web application.

See the Web Application

Essentially the App is a small but growing list of yoga poses. Each pose has its own page, and on each pose page, shows various web resources related to the pose. Building this small application gave me an opportunity to experiement with Views, Routes, and Services.

I am a student of Iyengar Yoga and web development. While considering how I could advance my knowledge and experience in both of these subjects, I decided to combine these two passions into a single effort by building myself an AngularJS application about yoga.

Iyengar Yoga is a unique system/methodology of yoga "characterized by main things": Timing, Sequencing, and Alignment. The goal of yoga, esssentially, is to access a state of union between and mind. Much like Web Frameworks like Bootstrap and AngularJS, which aim to systemize the development of web applications, the Iyengar Yoga methodology serves as a framework to systemtize the practice of Yoga and self realization.
